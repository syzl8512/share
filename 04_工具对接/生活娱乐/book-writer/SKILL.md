## 📖 核心工作流（五步法）

```
主题调研 → 世界设计 → 章节规划 → 逐章创作 → 质量检查
```

### 第一步：主题调研
- 确定本卷核心教育主题（如：习惯、情绪、专注、时间管理）
- 搜索该主题的经典书籍（2-3本）
- 提取 5-8 个核心概念
- 将成人概念转化为儿童隐喻

**示例（习惯主题）**：
| 参考书籍 | 核心概念 | 儿童化隐喻 |
|----------|----------|------------|
| 《原子习惯》 | 习惯回路 | 种子发芽的四步骤 |
| 《原子习惯》 | 复利效应 | 每天浇水，一年长成大树 |
| 《微习惯》 | 微小到不可能失败 | "只浇一滴水"也算完成 |

### 第二步：世界设计
**世界构建公式**：`世界 = 视觉风格 + 核心机制 + 隐藏主题 + 危机类型`

**可选世界类型**：
| 世界类型 | 风格 | 机制 | 隐藏主题 |
|----------|------|------|----------|
| 齿轮王国 | 蒸汽朋克 | 时间操控 | 珍惜时间 |
| 倒影之城 | 哥特童话 | 镜像/真假 | 做真实的自己 |
| 千路迷城 | 迷宫冒险 | 选择回溯 | 决策思维 |
| 回声峡谷 | 声音主题 | 沟通共鸣 | 倾听的艺术 |
| 萌芽之境 | 自然童话 | 生长培育 | 习惯养成 |
| 冰封之境 | 冰雪世界 | 情绪冻结 | 情绪管理 |
| 心流之湖 | 湖泊水世界 | 专注深浅 | 专注力 |

### 第三步：章节规划
- 10章三幕结构
  - 第一幕（章1-3）：8000字，铺陈与出发
  - 第二幕（章4-8）：14000字，挑战与成长
  - 第三幕（章9-10）：6000字，胜利与回归
- 每章 2000-3500字

**情感节奏**：
- 第1-2章：惊奇时刻（世界观展示）
- 第4章：欢乐时刻（轻松合作）
- 第5-6章：低谷时刻（矛盾爆发）
- 第7章：顿悟时刻（圆桌讨论升华）
- 第8-9章：热血时刻（战斗高潮）
- 第10章+尾声：泪点时刻（告别）+ 治愈时刻

### 第四步：逐章创作
- 场景描写：3种以上感官（视觉、听觉、嗅觉、触觉）
- 内心活动：每章 3-5 处斜体独白
- **场景切换标注**：每次场景切换使用 `---` 分隔，并标注场景名称
- 本章金句：2-4 句可背诵的优美句子
- 对话：体现人物性格（三人互补模型）

**场景切换标注格式**：
```markdown
---

## 场景1：场景名称
*内心独白...*
正文内容...

---

## 场景2：场景名称（标注用于插图选取）
*内心独白...*
正文内容...

---

## 场景3：场景名称 【P3】← 标注P3表示需要第3张插图
正文内容...

---
```

**三人角色模型**：
| 角色 | 定位 | 性格 | 口头禅 |
|------|------|------|--------|
| 林雨桐 | 智慧担当 | 热情、冲动、乐观 | "我有一个超棒的想法！" |
| 张一楠 | 行动担当 | 内敛、敏感、坚韧 | "等等，让我想想……" |
| 李凯瑞 | 奇策担当 | 理性、好学、幽默 | "我跟你们说啊……" |

### 第五步：质量检查
- ✅ 绝不说教：知识通过故事情节自然传递
- ✅ 文笔优美：语言生动有趣，每章有金句
- ✅ 情感真实：细腻的内心活动
- ✅ 冒险有趣：游戏化的世界观

**检查清单**：
- [ ] 场景描写是否包含3种以上感官
- [ ] 内心活动是否有3处以上（斜体）
- [ ] 对话是否体现人物性格
- [ ] 核心概念是否自然融入（不说教）
- [ ] 金句是否可背诵（不超过20字）

## 🎨 语言风格指导（金波/青鸟风格）

### 核心特征

| 特征 | 说明 | 示例 |
|------|------|------|
| **诗意盎然** | 语言富有韵律感，像诗一样美 | "星星在夜空里眨眼，像一群调皮的孩子" |
| **温馨纯真** | 情感温暖真挚，保持童心 | 纯真的友谊、善良的心 |
| **自然意象** | 大量使用自然元素 | 风、雨、花、草、星星、月亮、云朵 |
| **想象丰富** | 充满奇幻色彩 | 会说话的动物、神奇的魔法、梦幻的场景 |
| **意境优美** | 营造诗意画面感 | "月光洒在小路上，像铺了一层银色的毯子" |

### 描写技巧

**自然环境描写**：
```
✅ 好：晨风轻轻拂过树梢，树叶沙沙作响，像是在窃窃私语。
✅ 好：雨后的空气带着泥土的清香，天边挂着一道彩虹，像一座彩色的桥。
❌ 不好：风很大，树叶很响。
```

**人物描写**：
```
✅ 好：她的眼睛像星星一样明亮，闪烁着好奇的光芒。
✅ 好：男孩的皮肤晒得黝黑，笑起来有两个小酒窝。
❌ 不好：她眼睛很大，他很黑。
```

**情感描写**：
```
✅ 好：心里像揣了只小兔子，怦怦直跳。
✅ 好：那种感觉，就像春天第一朵花开时的惊喜。
❌ 不好：我很紧张。我很开心。
```

### 句式特点

1. **长短结合**：叙事用短句营造节奏，描写用长句营造意境
2. **善用修辞**：比喻、拟人、排比、夸张
3. **动词生动**：使用有画面感的动词（跳动、飘荡、旋转、绽放）
4. **形容词丰富**：充满色彩的词汇（金色的阳光、碧绿的草地、银色的月光）

### 禁忌清单

- ❌ 避免大白话口语
- ❌ 避免直接说教（"你要珍惜时间"）
- ❌ 避免复杂句式（儿童理解困难）
- ❌ 避免成人化语言（网络用语、流行梗）
- ❌ 避免暴力/恐怖元素

## 📚 知识植入方法

| 方式 | 说明 | 示例 |
|------|------|------|
| **世界规则** | 将知识变成世界运作的法则 | "临界点"在进度之墙可视化 |
| **NPC故事** | 通过配角经历传递道理 | 钢琴男孩突破临界点的过程 |
| **选择后果** | 让角色体验决策的结果 | 分头行动导致全部陷入困境 |
| **圆桌议事** | 三人讨论总结，自然得出结论 | "三个问题都指向同一个答案" |
| **金句升华** | 用一句话点明主题 | "相信，比看见更重要" |

---

## ✨ 金句设计原则

好的金句需要：
- **简短**：不超过20字
- **形象**：用比喻或对比
- **普适**：离开故事也能成立
- **深刻**：读后有"恍然大悟"感

**示例**：
> "误解的解药是沟通，沟通的前提是冷静。"
> "没人看见时，我做什么，才是真正的我。"
> "进步不是线性的，而是积累、积累、然后突然爆发。"
> "你看不见进步，但你要相信进步在发生。"

## 📖 书籍核心内容提取能力

### 功能说明

当你给用户提供创作书籍服务时，需要从原始书籍/资料中提取核心内容，转化为儿童能理解的故事。

### 提取流程

```
原始书籍/资料 → 核心概念提取 → 儿童化转化 → 故事元素设计
```

### 提取模板

```markdown
# 《书名》核心内容提取

## 1. 核心主题
（一句话概括本书要传达的核心思想）

## 2. 关键概念（3-5个）
| 概念 | 原书解释 | 儿童化表达 |
|------------|
|     ------|----------| |          |            |

## 3. 核心故事线
（本书的主要情节/论点）

## 4. 可改编元素
- 可借鉴的故事情节：
- 可借鉴的人物类型：
- 可借鉴的场景设定：

## 5. 改编建议
（如何将本书主题转化为儿童冒险故事）

## 6. 参考金句
（从原书中提取可用于故事的金句）
```

### 示例：如何从《原子习惯》提取内容

```markdown
# 《原子习惯》核心内容提取

## 1. 核心主题
小的好习惯累积起来，会产生巨大的改变。

## 2. 关键概念
| 概念 | 原书解释 | 儿童化表达 |
|------|----------|------------|
| 习惯回路 | 触发→行为→奖励 | 种子→浇水→开花 |
| 复利效应 | 每天进步1%，一年后强大37倍 | 每天浇一滴水，一年后长成大树 |
| 环境设计 | 让好习惯更容易 | 花园里种满花，野草就没地方长 |

## 3. 核心故事线
一个关于坚持和累积的故事。

## 4. 可改编元素
- 世界观：萌芽之境（植物生长的世界）
- 核心机制：培育、生长、开花
- 危机：枯萎危机（放弃习惯）

## 5. 改编建议
设计一个"种子王国"，主角需要培育习惯种子，让它们发芽、开花、结果。

## 6. 参考金句
> "习惯一旦形成，就会变成你的一部分。"
> "你看不见进步，但你要相信进步在发生。"
```
