# 🎨 书籍插图 Skill - 儿童冒险故事配图专用

> 用于：为书籍创作生成场景插图提示词、配图生成、文章匹配

---

## 一、插图工作流

```
章节内容 → 场景标注 → 提示词生成 → 图片生成 → 文章匹配 → 排版输出
```

---

## 二、场景选取规则

### 2.1 每章选取原则

| 章节位置 | 插图数量 | 场景类型 |
|----------|----------|----------|
| 第1-3章 | 4-5张/章 | 世界观建立、人物出场、重要情节 |
| 第4-8章 | 5-6张/章 | 冒险场景、情感高潮、关键转折 |
| 第9-10章 | 5-6张/章 | 终极对决、告别、治愈结尾 |
| 番外篇 | 3-4张/章 | 轻松场景、趣味互动 |

### 2.2 场景选取标准

**必选场景**：
- 🔥 章节开篇场景（建立世界观）
- 🎯 核心情节场景（推动故事）
- 💫 情感高潮场景（内心活动/金句时刻）
- 🌟 主视觉场景（全章核心画面的跨页插图）

**可选场景**：
- 🔄 场景转换（世界切换）
- 🗣️ 对话场景（三人互动）
- 🏞️ 环境展示（世界细节）

### 2.3 场景标注格式

在章节文档中，使用以下格式标注需要插图的场景：

```markdown
---

## 场景标注区

### 场景1：场景名称 【P1】
**位置**：第X章/第Y段
**内容描述**：场景的具体画面
**情绪**：惊奇/欢乐/低谷/顿悟/热血/泪点/治愈

### 场景2：场景名称 【P2】
**位置**：...
**内容描述**：...
**情绪**：...

---
```

## 四、图片生成

### 4.1 推荐的AI绘画工具

| 工具 | 优点 | 适用场景 |
|------|------|----------|
| Midjourney | 画风精美，一致性好 | 正式插图 |
| DALL-E 3 (ChatGPT) | 理解力强 | 快速生成 |
| Leonardo.ai | 免费，效率高 | 批量生成 |
| Adobe Firefly | 商用安全 | 正式出版 |

### 4.2 输出规格

| 参数 | 推荐值 |
|------|--------|
| 比例 | 3:4（竖版书籍）、16:9（跨页） |
| 分辨率 | 2048x2732 或更高 |
| 格式 | PNG（透明背景）、JPG（正式出版） |
| 风格 | 水彩风格、宫崎骏风格 |

### 4.3 生成后存储结构

```

│   ├── 第二章_插画提示词.md
```

## 七、批量替换（主角/配角/场景）

### 7.1 替换变量系统

在提示词中使用变量，便于批量替换：

```markdown
# 变量定义
{{主角1}} = 林雨桐
{{主角1_英文}} = Yutong
{{主角1_发型}} = 齐刘海，双辫子，辫子上系蝴蝶结丝带
{{主角1_特征}} = 大而明亮的眼睛，好奇的表情

{{主角2}} = 张一楠
{{主角2_英文}} = Yinan
...

{{世界名}} = 书页之城
{{世界风格}} = 书页构建的城市，金色书页飞翔
```

### 7.2 批量替换模板

**主角替换**：
```python
# 替换主角
replacements = {
    "林雨桐": "新主角名",
    "Yutong": "NewYutong", 
    "齐刘海，双辫子": "新发型描述",
    # ...
}
```

**场景替换**：
```python
# 替换场景
replacements = {
    "书页之城": "新世界名",
    "金色书页": "新视觉元素",
    # ...
}
```

### 7.3 一键替换命令

```bash
# 批量替换主角名
sed -i '' 's/林雨桐/新主角名/g' *.md

# 批量替换世界名
sed -i '' 's/书页之城/新世界名/g' *.md

# 批量替换提示词中的角色描述
find . -name "*.md" -exec sed -i '' 's/齐刘海/新发型/g' {} \;
```

## 八、设计理念（Design Philosophy）

### 8.1 视觉风格

** Literary Luminescence（文字光辉）**：
- 文字作为发光的生命体
- 纸张和排版作为神圣材料
- 文学与建筑的交汇

### 8.2 色彩体系

| 色彩 | 含义 | 适用场景 |
|------|------|----------|
| 金色 | 智慧、魔法 | 主角光环、重要时刻 |
| 墨蓝色 | 深邃、神秘 | 背景、夜晚 |
| 纯白 | 纯真、可能 | 留白、想象空间 |
| 暖琥珀 | 光与纸的交汇 | 温馨场景 |

### 8.3 构图原则

- **层次感**：前景、中景、背景分明
- **垂直运动**：暗示上升、成长、知识积累
- **呼吸空间**：密集与开放的对比
- **细节丰富**：经得起仔细观看
