# Book Adventure Writer Skill

> 儿童冒险书籍创作技能 - 基于《史书活字秘境》方法论

## 使用场景

当用户要求创作儿童冒险书籍时使用此技能，包括：
- 创建新的冒险书籍系列
- 制定书籍大纲
- 创作新章节
- 设计世界观和关卡

## 创作方法论（必须遵循）

### 1. 基础设定模板

```
卷名：XXX秘境·XXX
主题：XXX（核心道理）
适配IP：双人合作冒险（双影奇境/双人成行式）
语言风格：温柔治愈、画面感强、道理藏情节、对话像姐妹谈心、不低幼有文学质感
```

### 2. 主角设定模板

| 角色 | 身份 | 特点 | 能力 |
|------|------|------|------|
| **小主角** | 小学生 | 感知者、好奇活泼 | 视角入镜——走进场景，共情人物情绪 |
| **大主角** | 中学生 | 沉稳、能力强 | 史鉴解意——提炼规律、连接古今 |

### 3. 世界观模板

```
这是一个活的XXX秘境：书页为大地、活字为生灵、XXX为层楼、XXX为独立卷轴、XXX可实景重演。
世界正在褪色失忆：XXX。只有双主角以三视角共读才能重燃光芒。

核心规则：看见≠读懂，静心才亮，双人同行，视角互补
```

### 4. 敌人体系（温和成长向）

| 敌人 | 形态 | 对应学习痛点 |
|------|------|-------------|
| XXX雾 | XXX | XXX |
| XXX影 | 单一视角、贴标签 | 人云亦云 |
| XXX虫（小BOSS） | 吞噬XXX | 觉得XXX枯燥无用 |
| XXX之影（最终BOSS） | 冰冷知识点 | 否定XXX价值 |

## 100%落地检查清单

每次创作前必须检查：

- [ ] 主角设定（双人能力互补）
- [ ] 世界观完整（秘境结构+规则）
- [ ] 敌人体系（4类敌人）
- [ ] 核心主旨明确（一句话说明白）
- [ ] 用户核心要求全部落地
- [ ] 10章结构完整
- [ ] 不说教（道理藏情节）
- [ ] 语言风格统一（温柔治愈）

---

## 写作规范（每章必须包含）

| 项目 | 要求 |
|------|------|
| 心理描写 | ≥3处 |
| 对话 | ≥2处 |
| 情节冲突 | 有波澜，引人入胜 |
| 感官场景 | ≥3处（视/听/嗅/触） |
| 结尾悬念 | 必须留悬念 |
| 金句 | ≥2句可背诵 |

### 禁止使用

- 说教词汇（应该、必须、记住）
- 直接讲道理（通过故事展现）
- 成人化表达
- 流水账叙述
