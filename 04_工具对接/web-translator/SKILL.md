# Skill: Web Translator

## 描述
网页双语对照翻译，类似沉浸式翻译插件。获取网页内容并提供中英对照翻译。

## 触发
用户说"翻译网页"、"双语阅读"、"翻译这个页面"。

## 工具
- web_fetch: 获取网页内容
- AI: 进行翻译和对照排版

## 流程
1. 用户提供URL
2. 用web_fetch获取内容
3. AI翻译并保持原文对照格式
4. 返回双语对照结果

## 示例
```
用户: 翻译 https://example.com
→ 获取内容 → 中英对照排版 → 返回

用户: 帮我读这个英文页面
→ 获取 → 翻译 → 解释要点
```

## 格式
```
原文: This is a sample text.
译文: 这是一段示例文本。
```

## 注意
- 长内容可能需要分页
- 保持原文段落结构
- 可以指定翻译风格（直译/意译）
