# Skill: 文件归档管理器 (File Archive Manager)

## 描述
规范文件存放流程，确保源文件、过程文件、成品文件、导出文件分类存储。每次生成成品文件后必须执行归档流程。

## 触发
当需要保存或归档文件时使用。

## 文件存放规则

### 基础目录
```
/Users/agent/Documents/syzl8512/00_收件箱/
├── 样本分析_源文件/      # 原始输入文件
├── 样本分析_过程文件/    # 分析过程中的markdown文件（可删除）
├── 样本分析_成品文件/    # 最终成品(markdown)
└── 样本分析_导出文件/   # 导出的Word/PDF文件
```

### 文件命名规则
| 类型 | 命名格式 | 示例 |
|------|----------|------|
| 源文件 | 原文件名 | 湖北省政府2026年一号文件.md |
| 过程文件 | 原名_分析类型 | 湖北省政府2026年一号文件_16维分析.md |
| 成品文件 | 原名_成品类型_版本 | 湖北省政府2026年一号文件_16维分析_v1.md |
| 导出文件 | 原名_成品类型_日期 | 湖北省政府2026年一号文件_16维分析_20260225.docx |

### 文件类型说明
| 类型 | 格式 | 存放目录 |
|------|------|----------|
| 源文件 | 原始PDF/Word/网页/笔记 | 样本分析_源文件 |
| 过程文件 | 分析过程中的md | 样本分析_过程文件 |
| 成品文件 | 最终markdown | 样本分析_成品文件 |
| 导出文件 | Word/PDF | 样本分析_导出文件 |

## 操作流程（每次成品生成后必须执行）

### Step 1: 确认源文件
- 检查源文件是否在 `样本分析_源文件/`

### Step 2: 归档过程文件
- 将分析过程文件存放到 `样本分析_过程文件/`

### Step 3: 归档成品文件
- 将最终成品存放到 `样本分析_成品文件/`

### Step 4: 导出（如需要）
- 转换为Word/PDF
- 存放到 `样本分析_导出文件/`

## 重要提示
- 每次生成成品文件后**必须执行归档流程**
- 过程文件可在确认成品后删除以节省空间
- 导出文件需标注日期版本
