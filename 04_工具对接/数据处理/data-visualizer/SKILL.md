# Skill: Data Visualizer

## 描述
数据可视化生成，类似Flourish。输入数据生成动态图表。

## 触发
用户说"可视化"、"做图表"、"生成图表"、"数据展示"。

## 工具
- exec: 调用外部API
- AI: 处理数据生成配置

## 流程
1. 用户提供数据（CSV/Excel/描述）
2. 选择图表类型
3. 生成可视化配置或调用API
4. 返回图表链接或嵌入代码

## 支持图表类型
- 条形图/柱状图
- 折线图
- 饼图
- 地图
- 散点图
- 动态排名图

## 示例
```
用户: 把这些数据做成图表
→ 数据: A:100, B:200, C:150
→ 生成条形图 → 返回链接

用户: 做一个世界地图展示各国GDP
→ 调用 Flourish API → 返回可视化
```

## 注意
- 需要Flourish API或本地图表库
- 可以指定配色、动画效果
