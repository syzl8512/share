# 核心工作流详情

## 3.1 /学习管家Plus - 智能素材处理

**功能**：统一入口的智能素材处理中枢

**输入处理**：
- 作业/试卷图片 → MinerU API (OCR)
- 会议音频 → 会议洞察技能
- 政策PDF → PDF处理技能
- 其他图片 → ZAI MCP
- 视频 → ZAI MCP
- 网页链接 → WebFetch

**分类逻辑**：
- 女儿教育 → 04_教育/学校教育/{作业记录|成绩记录}
- 会议内容 → 02_工作/{会议纪要|政策与趋势|日常报告}
- 工作政策 → PDF深度解析 → 产品匹配
- 学习资料 → 03_生活/学习记录/

## 3.2 /书籍创作 - 儿童成长书籍

**五步流程**：需求澄清 → 世界设计 → 章节规划 → 逐章创作 → 质量检查

**核心原则**：
1. 绝不说教 - 知识通过体验传递
2. 文笔优美 - 每章有可背诵金句
3. 情感真实 - 细腻内心活动
4. 冒险有趣 - 游戏化世界观

**规范**：10章三幕结构 (28000字)，单章 2000-3500字
