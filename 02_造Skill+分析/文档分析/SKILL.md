# Skill: Document Summarizer

## 描述
对PDF、文档、网页内容进行智能摘要、问答、知识点提取。类似ChatPDF的能力。

## 触发
用户说"总结"、"摘要"、"提取要点"、"问答"、"出题"时使用。

## 工具
- read: 读取文件
- web_fetch: 获取网页内容
- AI: 用MiniMax进行理解和分析

## 流程
1. 用户上传文件或提供URL
2. 用read或web_fetch获取内容
3. 根据用户需求（总结/问答/出题）构造提示词
4. 调用AI处理
5. 返回结果

## 示例
```
用户: 帮我总结这个PDF的重点
→ 读取文件 → 提取关键信息 → 返回摘要

用户: 这个文档里说了什么?
→ 读取文件 → 理解内容 → 简要回答
```

## 注意
- 长文档可以分块处理
- 支持PDF、TXT、Markdown等格式
