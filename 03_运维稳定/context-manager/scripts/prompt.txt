# 会话摘要提取提示词

请总结以下会话记录，提取关键信息。

## 提取规则

### 必须保留
1. **关键决策** - 用户明确的要求、偏好、禁止事项
2. **待办事项** - 未完成的任务、后续步骤
3. **上下文** - 当前项目/任务的背景信息
4. **配置变更** - 系统配置的修改
5. **重要结论** - AI 给出的最终答案、方案

### 应该丢弃
1. **调试过程** - 失败的尝试、错误信息
2. **重复内容** - 多轮相似的讨论
3. **中间推导** - 思维过程、试错记录
4. **工具调用** - 纯技术性的命令执行日志

## 输出格式

```markdown
## 会话摘要 [日期]

### 关键决策
- xxx

### 待办事项
- [ ] xxx

### 当前上下文
- 项目：xxx
- 进度：xxx

### 重要结论
- xxx
```
