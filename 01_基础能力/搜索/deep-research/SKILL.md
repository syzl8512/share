# Skill: Deep Research

## 描述
深度研究助手，类似Perplexity/DeepResearch。对任意主题进行深入研究并生成报告。

## 触发
用户说"研究一下"、"深度分析"、"调研"、"写报告"。

## 工具
- web_search: 搜索相关信息
- web_fetch: 获取详细内容
- AI: 综合分析生成报告

## 流程
1. 用户给出研究主题
2. 搜索相关权威来源
3. 获取关键页面内容
4. AI综合分析
5. 生成结构化研究报告

## 报告结构
```
# {主题}深度研究报告

## 一、概述
## 二、核心发现
## 三、详细分析
## 四、数据来源
## 五、结论与建议
```

## 示例
```
用户: 研究一下AI在教育领域的应用
→ 搜索相关论文/报告 → 获取详情 → 综合分析 → 返回万字报告

用户: 调研新能源汽车市场趋势
→ 搜索数据 → 分析趋势 → 生成报告
```

## 注意
- 研究时间可能较长（5-10分钟）
- 会标注信息来源
- 可以指定研究深度（简要/详细/全面）
